# Живая история: малые музеи Ставрополья

Мини-приложение для мессенджера МАХ. Позволяет жителям и гостям края узнавать о малых музеях, отмечать посещения, подписываться на события. Сотрудники музеев могут управлять контентом через админ-панель.

## Функции
- Главная: карточки музеев с фото, описанием, ссылкой на сайт.
- Яндекс.Карта с метками музеев.
- События (лента с фильтром по подпискам).
- Мой паспорт: ручная отметка посещений, прогресс-бар.
- Админ-панель (пароль admin123): добавление/редактирование/удаление музеев и событий.

## Развертывание
1. Установите Python 3.10.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python app.py`
4. Откройте http://localhost:5000

## Деплой на Render
- Создайте Web Service, укажите репозиторий.
- Build Command: `pip install -r requirements.txt`
- Start Command: `python app.py`

## Технологии
- Flask, SQLite, JavaScript, Яндекс.Карты
